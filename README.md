# MSCS532 – Assignment 1: Insertion Sort

## Objective
Sort a user-provided list of numbers in **monotonically decreasing order** using the **Insertion Sort algorithm**.

## Python Version
- Python 3.8 or higher

## How to Run?

### Clone the project with following command
```bash
git clone https://github.com/sranabhat51358/MSCS532_Assignment1.git
cd MSCS532_Assignment1
```
### Run the python file using following command
```bash
python insertion_sort.py
```
### Follow the prompts to provide input:
1. Please enter the number of elements you want to sort:
2. Enter element {i+1}:

## Input Example
```
Please enter the number of elements you want to sort: 6
Enter element 1: 13
Enter element 2: 17
Enter element 3: 9
Enter element 4: 4
Enter element 5: 64
Enter element 6: 24
```

## Output Example
```
Provided list for sorting:  [13, 17, 9, 4, 64, 24]
Sorted List in Monotonically Decreasing Order:  [64, 24, 17, 13, 9, 4]
```
